# 3d-molecular-sketcher

3D Molecular Sketcher is a lightweight molecular visualization application that renders realistic 3D molecular models 

Users can:

- Search molecules instantly
- Visualize atoms and bonds in 3D
- Rotate, zoom, and pan structures
- Toggle display options
- View molecular information and statistics
- Explore built-in molecules from a predefined database

## ✨ Features

### 🔬 Molecular Visualization
- Real-time 3D rendering using Three.js
- Atom-based sphere representation
- Single, double, and triple bond rendering
- Automatic molecule scaling and centering

### 🎨 Scientific Color Coding
- Standard CPK color scheme
- Element-specific atomic radii
- Dynamic legend generation

### 📚 Built-in Molecule Database
Includes 30+ predefined molecules 

### 🎯 Interactive Controls
- Mouse drag → Rotate molecule
- Right drag → Pan view
- Scroll wheel → Zoom
- Touch support for mobile devices
- Auto-rotation mode

### 📊 Molecule Information Panel
Displays:

- Molecule name
- Molecular formula
- Atom count
- Bond count
- Unique elements
- Description
